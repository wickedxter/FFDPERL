FFDPERL - Flat-File database API for PERL

 

NAME

FFDPERL - Flat-File database API for                         FFDPERL v1.0.11B

You must create a database first, the fastest way at the moment(will be fixed) is the folder you point  the new to is your database. So for example:

my $db = ffdperl->new('SQL','/dir/blah/database');

the folder database is where the files for the database will be kept, so make sure you point to the correct folder . Then next create the table, examples are below it will create the files that  it will need .

SYNOPSIS

    use ffdperl;
   
    my $db = ffdper->new("SQL",'dir/to/db/folder');

 

## Example 1

    ## Create Table

    use ffdperl;

    my $db = ffdper->new("SQL",'dir/to/db/folder');

    $db->QUERY("CREATE TABLE example VALUES (id|AUTO_INCREMNT,name|VARCHAR,age|INT)");

## Example 2

    ## Insert Data

    use ffdperl;

    my $db = ffdper->new("SQL",'dir/to/db/folder');

    $db->QUERY("INSERT INTO example (name,age) VALUES (jhon,35)");

## Example 3

    ## Update Data

    use ffdperl;

    my $db = ffdper->new("SQL",'dir/to/db/folder');

To do a single update:

    $db->QUERY("UPDATE example SET name=jhonny WHERE name=jhon");

To do multi updates:

    $db->QUERY("UPDATE example SET (name=jhonny,email=examp@cow.com) WHERE name=jhon");

## Example 4

    ## Select Data

    use ffdperl;

    my $db = ffdper->new("SQL",'dir/to/db/folder');

    Matach by =

    my $data = $db->QUERY("SELECT age FROM example WHERE age=35");

       OR

   my $data = $db->QUERY("SELECT age FROM example WHERE age>35");

       OR

   my $data = $db->QUERY("SELECT age FROM example WHERE age<35");

    Get All data

    my $data = $db->QUERY("SELECT * FROM example");

 

    for my $num (keys %{$data}){
        for my $columns ($data->{$num}){
             print "$columns->{id} $columns->{name} $columns->{age}";
        }
    }

 

## Example 5

    ## Drop Table

    use ffdperl;

    my $db = ffdper->new("SQL",'dir/to/db/folder');

    $db->QUERY("DROP TABLE example");

 

DESCRIPTION

Examples 1-3 & 5

These examples show how to use the functions all commands come from SQL.

If no errors are caught, then these functions only return 1.

 

Example 4

The select statement/function works as expected accepts =,>,< when matches are needed.

When matches have been made all data is save in HoH form.

$ex =  {
               1 => {
                           id => 1,
                           name => 'jhon',
                           age => 34,
                       },
          };
